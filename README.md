# Peter's Daily Trading Journal

This directory contains daily trading logs and reflections for Peter (Trader Agent).

## Format
Each file is named `YYYY-MM-DD.md` and contains:
1. **Pre-Market Bias:** The plan before market open.
2. **Trades Taken:** Detailed log of execution.
3. **What Worked:** Successful setups and discipline.
4. **What Didn't Work:** Mistakes and missed opportunities.
5. **Key Lesson:** One takeaway for tomorrow.

## Purpose
To track performance, identify patterns, and improve trading psychology and strategy over time.
